What This Project Does
End-to-end bank transaction categorization, reconciliation, and posting tracker

Each week, the RCM team exports banking data from CitiBank (commercial accounts), PNC (lockbox check deposits), and account statements. The bank_pivot.py script processes all three data sources and generates an interactive HTML dashboard where the posting team can:

  • View all incoming deposits categorized by type (Card, EFT, Lockbox, Medicaid)
  • Track EOB receipt status for each transaction
  • Mark transactions as Posted / Not Posted / Partial in Open Dental
  • Add remarks for follow-up or exceptions
  • Reconcile bank statement totals against transaction-level data

The two posting tracks run in parallel: PPO (commercial insurance, lockbox checks) and Medicaid (Skygen, DentaQuest, Avesis).

Weekly Workflow
Complete step-by-step process to run every Monday/Tuesday
1

Export CitiBank Build Report

Log into CitiBank Commercial Banking. Navigate to Payments & Transfers > Transaction Search. Filter by the prior week (Mon-Sun).

  • Export PPO account (6881784489) → save as MM.DD to MM.DD PPO.csv
  • Export Medicaid account (6881784534) → save as MM.DD to MM.DD Medicaid.csv
  • Place both files in Build Report/{Month}/
2

Export PNC Lockbox Data

Log into PNC ActivePay / Lockbox portal. Download transaction detail for the week.

  • Save as MM.DD-MM.DD.csv
  • Place in LockBox/{Month}/
3

Export Bank Statements (CCB)

Download checking account CSVs from CitiBank for both accounts.

  • PPO: CCB_CHECKING_6881784489_*.csv
  • Medicaid: CCB_CHECKING_6881784534_*.csv
  • Place in Bank General/{Month}/
4

Update Script Paths

Open bank_pivot.py and update the file paths to match the new week's filenames:

# Line 5 — PPO Build Report
df_ppo = pd.read_csv("Build Report/{Month}/MM.DD to MM.DD PPO.csv")

# Line 7 — Medicaid Build Report
df_medicaid = pd.read_csv("Build Report/{Month}/MM.DD to MM.DD Medicaid.csv")

# Line 276 — LockBox
lb = pd.read_csv("LockBox/{Month}/MM.DD-MM.DD.csv")

# Lines 368-369 — Bank General
bg_ppo = load_bank_general("Bank General/{Month}/CCB_CHECKING_6881784489_*.csv", "PPO")
bg_med = load_bank_general("Bank General/{Month}/CCB_CHECKING_6881784534_*.csv", "Medicaid")
5

Run the Script

cd /Users/Admin/Desktop/Claude/BANK
python bank_pivot.py

This generates Bank_Transaction_Pivot.html in the same directory.

6

Review & Post

Open the generated HTML dashboard. The posting team uses it to track EOBs, mark transactions as posted in Open Dental, and add remarks for follow-up.

7

End-of-Week Reconciliation

Verify all EFTs are marked as "Posted". Follow up on any "Partial" or unmarked items. Flag discrepancies between lockbox totals and bank statements.

Classification Logic
deposit_sources = ['BANKCARD-8740', 'MERCHANT BANKCD', 'SYNCHRONY BANK']
lockbox_sources = ['PNC-ECHO']

def classify(row):
    name = row['From Account Name']
    if name in deposit_sources: return 'Deposits'
    elif name in lockbox_sources: return 'Lockbox'
    else: return 'EFT'

Use this checklist each week to ensure nothing is missed

  • CitiBank Medicaid Build Report exported
